# rules-us-ks

Kansas tax and benefit rules encoded in Cosilico DSL.

## Structure

```
rules-us-ks/
├── statutes/       # Kansas Statutes Annotated (K.S.A.)
│   └── 79/         # Chapter 79: Taxation (when added)
└── regulations/    # Kansas Administrative Regulations (K.A.R.)
    └── 92/         # Article 92: Department of Revenue (when added)
```

## Sources

- **Statutes**: [Kansas Legislature](http://www.kslegislature.org/li/statute/)
- **Regulations**: [Kansas Secretary of State](https://www.sos.ks.gov/pubs/kar.aspx)
